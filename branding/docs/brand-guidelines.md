## Visual Identity

### Logo

#### Primary Logo
The RefData Hub logo represents the transformation of raw, scattered data into organized, structured canonical values.

**Design Elements:**
- **Left side:** Chaotic geometric shapes (raw data)
- **Center:** Gradient transition (transformation)
- **Right side:** Aligned data rows (canonical values)
- **Overall shape:** Rounded container with modern aesthetic

#### Logo Variants
| Variant | File | Use Case |
|---------|------|----------|
| Primary (Full Color) | `refdata-hub-primary.svg` | General use, presentations |
| Dark Mode | `refdata-hub-primary-dark.svg` | Dark backgrounds |
| Light Mode | `refdata-hub-primary-light.svg` | Light backgrounds |
| Monochrome | `refdata-hub-mono.svg` | Print, one-color applications |
| Icon Only | `refdata-hub-icon.svg` | Small spaces, app icons |
| Horizontal | `refdata-hub-horizontal.svg` | Headers, wide layouts |
| Stacked | `refdata-hub-stacked.svg` | Compact vertical spaces |

#### Logo Usage Rules
- **Minimum Size:** 24px height for digital, 0.5" for print
- **Clear Space:** Maintain 1x logo height around all sides
- **Prohibited:**
  - Do not stretch or distort proportions
  - Do not rotate or flip
  - Do not add filters or effects
  - Do not change colors (except monochrome version)
  - Do not place on low-contrast backgrounds
